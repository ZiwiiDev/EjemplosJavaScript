El Objeto History

Este objeto contiene la lista de las URLs visitadas por el usuario.
Por desgracia no es completamente estándar.
Veamos las propiedades y los métodos.
El resto de los métodos los veremos para trabajar con ajax.

El Objeto Location

hash establece o devuelve la parte (#) de una URL
host Establece o devuelve el nombre y puerto de una URL
hostname Establece o devuelve el nombre de una URL
href Establece o devuelve una URL completa.
y puerto de una URL
origin Devuelve el protocolo, nombre de host pathname Establece o devuelve una ruta de una URL
port Establece o devuelve el número de puerto de una URL
protocol Establece o devuelve el protocolo de una URL
search Establece o devuelve la cadena de consulta parte de una URL
assign() Carga un nuevo documento
reload() Recarga el nuevo documento
replace() Reemplaza el documento actual con uno nuevo
La diferencia entre replace y assign es que uno elemina la historia anterior y el otro no.
