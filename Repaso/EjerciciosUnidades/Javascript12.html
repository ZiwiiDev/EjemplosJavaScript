Objeto Number , o sea, ¡Más Mates!

Muchas veces hay que visualizar un número menor de decimales. Para ello podemos utilizar un metodo del número llamado toFixed(número de decimales);
Se hace un redondeo. También pueden convertirse a notación exponencial. toExponential(Número de digitos a conservar);
Ahora con el método toPrecision( precisión) vamos a indicar cuantos dígitos vamos a utilizar.
Como habréis comprobado solo se cuenta digitos distintos de cero a la izquierda.
Y por último el método que tienen todas las clases toString(). Convierte el número a cadena.
Tambien podemos declarar los números 100% como objetos. Para eso utilizaremos
var numero= new Number (12.3);
